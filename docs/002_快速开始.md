# 快速开始

## 在新项目中使用

### 1. 添加依赖

```json
{
  "dependencies": {
    "@usethink/cf-core": "file:../../packages/cf-core"
  }
}
```

### 2. 配置 tsconfig.json

```json
{
  "compilerOptions": {
    "paths": {
      "@usethink/cf-core/*": ["../../packages/cf-core/src/*"]
    }
  }
}
```

### 3. 最小入口

```ts
// src/index.ts
import { bootstrap } from "@usethink/cf-core/bootstrap";
import * as schema from "./db/schema";

export default bootstrap({
  schema,
  registerRoutes: (api) => {
    api.get("/hello", (c) => c.json({ ok: true, message: "Hello!" }));
  },
  pageRoutes: { "/admin": "/admin.html" },
});
```

### 4. 数据库 Schema

```ts
// src/db/schema.ts
export { systemConfig, adminAuditLogs, apiKeys } from "@usethink/cf-core/db/schema";

import { sqliteTable, text, integer } from "drizzle-orm/sqlite-core";

export const products = sqliteTable("products", {
  id: text("id").primaryKey(),
  name: text("name").notNull(),
  priceCents: integer("price_cents").notNull(),
});
```

## 按需引入功能模块

| 需要什么 | 导入路径 |
|---------|---------|
| 支付（支付宝） | `@usethink/cf-core/features/payment` |
| 邮件（Resend） | `@usethink/cf-core/features/email` |
| Webhook 通知 | `@usethink/cf-core/features/webhook` |
| 智能路由 | `@usethink/cf-core/features/thompson-router` |
| 防刷检测 | `@usethink/cf-core/features/anti-abuse` |
| Telegram 集成 | `@usethink/cf-core/features/telegram-miniapp` |

## 运行测试

```bash
cd packages/cf-core
npm test
```
