# @eforge/cf-core 架构总览

## 定位

Cloudflare Workers 共享内核，为批量产品复制提供统一基础设施。

## 三层结构

```
┌─────────────────────────────────────────┐
│  features/ — 可选功能模块（按需引入）      │
│  payment / email / webhook / ...         │
├─────────────────────────────────────────┤
│  src/ — 核心模块（所有项目必须）           │
│  http / security / db / bootstrap / ...  │
├─────────────────────────────────────────┤
│  Cloudflare Workers + Turso + Drizzle    │
└─────────────────────────────────────────┘
```

## 设计原则

1. **泛型优先** — 所有函数使用泛型 `Context<E>` 而非特定 `AppEnv`
2. **零耦合** — 模块间仅通过导入引用，不依赖全局状态
3. **Web Crypto** — 所有加密操作使用 Workers 原生 API
4. **Free 友好** — Cache API 优先、Isolate 级缓存、惰性清理

## 文件结构

```
packages/cf-core/
├── src/                          # 核心（19 个模块）
│   ├── index.ts                  # 统一导出
│   ├── types.ts                  # 公共类型
│   ├── http.ts                   # ok/fail/maskContact/csvEscape
│   ├── security.ts               # sha256/IP哈希/Turnstile/安全响应头
│   ├── rate-limit.ts             # Memory/KV/DB 三后端限流
│   ├── cache.ts                  # Workers Cache API
│   ├── idempotency.ts            # 原子 UPSERT 幂等
│   ├── audit.ts                  # 审计日志
│   ├── config.ts                 # SystemConfig 热生效
│   ├── bootstrap.ts              # Worker 入口工厂
│   ├── error.ts                  # 错误分类 + 指数退避
│   ├── logger.ts                 # 结构化日志
│   ├── crypto.ts                 # AES-256-GCM
│   ├── db/                       # 数据库层
│   ├── auth/                     # JWT + PBKDF2
│   └── middleware/               # Admin + API Key 认证
├── features/                     # 可选功能模块（6 个）
│   ├── payment/                  # 支付抽象 + 支付宝
│   ├── email/                    # Resend 邮件
│   ├── webhook/                  # Webhook 通知
│   ├── thompson-router/          # Thompson Sampling 路由
│   ├── anti-abuse/               # 防刷检测
│   └── telegram-miniapp/         # Telegram 前端集成
├── tests/                        # 核心测试（53 个）
├── features/*/tests/             # 功能模块测试（28 个）
└── docs/                         # 文档
```

## 导入方式

```ts
// 核心（所有项目）
import { ok, fail } from "@eforge/cf-core/http";
import { sha256, verifyTurnstile } from "@eforge/cf-core/security";
import { bootstrap } from "@eforge/cf-core/bootstrap";

// 功能模块（按需）
import { AlipayProvider } from "@eforge/cf-core/features/payment";
import { EmailService } from "@eforge/cf-core/features/email";
import { WebhookService } from "@eforge/cf-core/features/webhook";
```
